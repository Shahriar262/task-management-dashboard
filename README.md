# Donezo Dashboard

A modern task management dashboard built with React, JWT authentication, and REST API integration.

## 🔗 Live Demo
[https://task-management-dashboard-shahriar262.netlify.app/](https://task-management-dashboard-shahriar262.netlify.app/)

## 🛠 Tech Stack
- React
- React Router
- Axios
- Tailwind CSS
- JWT Authentication

## ✨ Features
- Secure Login (Token-based)
- Protected Routes
- Dynamic API Data
- Responsive UI
- Modern Sidebar Layout
